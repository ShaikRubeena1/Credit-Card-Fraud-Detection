# 💳 Credit Card Fraud Detection

## 📌 Project Overview

Credit card fraud is one of the major challenges faced by financial institutions. Fraudulent transactions cause significant financial losses 
and impact customer trust.

This project aims to build a Machine Learning model that can accurately identify fraudulent credit card transactions and
help financial organizations detect suspicious activities in real time.

---

## 🎯 Business Problem

Financial institutions process millions of transactions every day. Detecting fraudulent transactions manually is nearly impossible.

The objective of this project is to:

- Identify fraudulent credit card transactions.
- Minimize financial losses.
- Improve transaction security.
- Build an efficient fraud detection system using Machine Learning.

---

## 📊 Dataset Information

The dataset contains credit card transactions made by European cardholders.

### Features

- Time
- Amount
- V1 to V28 (anonymized features)
- Class

Target Variable:

- Class = 0 → Genuine Transaction
- Class = 1 → Fraudulent Transaction
